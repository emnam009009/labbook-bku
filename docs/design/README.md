# Design System Documentation

This folder contains detailed specifications for LabBook BKU UI/UX design system.

## Master document

See `/DESIGN.md` at repo root.

## Detail files (added in subsequent rounds)

- `TOKENS.md` — Color, typography, spacing tokens (companion to /DESIGN.md)
- `COMPONENTS.md` — Component patterns (buttons, inputs, tables, sidebar, sidetab)
- `PATTERNS.md` — Layout patterns (anti-card, command palette, lab mode)
- `MIGRATION.md` — Step-by-step migration from current UI to new design

## Reading Order

1. `/DESIGN.md` (root, master philosophy)
2. `TOKENS.md` (foundation values)
3. `COMPONENTS.md` (building blocks)
4. `PATTERNS.md` (composition)
5. `MIGRATION.md` (how to redo current pages)
